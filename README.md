# Foodgram - Продуктовый помощник  
  
## краткое описание:  
Сервис **Foodgram** - сайт, на котором пользователи публикуют рецепты, могут добавлять чужие рецепты в избранное и подписываться на публикации других авторов.  
  
## технологии в проекте :
Django
